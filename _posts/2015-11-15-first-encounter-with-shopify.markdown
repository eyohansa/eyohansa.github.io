Recently, I was given a chance to work on a Shopify site through a freelancing offer. It was my first time working with a Shopify site so I took my time reading through its Liquid template engine. The engine itself is quite straightforward and it was really easy to understand. I used the [cheat sheet]{http://cheat.markdunkley.com) by Mark Dunkley a lot to help me going through the project as soon as possible.
 
This job was mainly on the design part, which means a lot of CSS tinkering, and what an experience that was. For the inexperienced me, dealing with CSS can be frustrating. Multi-layered divs, involving containers and elements, just to get those pixel-perfect alignments. In addition, I had to use Shopify's web text editor the whole time. A lot of times, it has weird tab behavior in which it keeps changing between two and four whitespaces. It causes me to miscount tag-pairs.

Despite the frustrations, I did learn a lot of new things. Albeit inexperienced, I did get the design implemented correctly and it was even responsive! Before this, I didn't really have any chance to apply my media queries knowledge. I also got to use the dev console for inspecting dimensions instead of just scripts like usual. I also got to read some SASS codes from the shop's default theme. It was quite easy to understand and I might be picking it up to learn really soon.

Overall, I must say it's a nice feeling to get a project done amidst a lot of on-going projects that I'm working on.
